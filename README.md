Student Study Tracker

A Java-based desktop application that helps students track their study activities, manage records, and monitor progress using a simple and interactive UI.

🚀 Features
➕ Add new students
📝 Record daily study logs (hours & stress level)
👀 View students and study logs in a table format
✏️ Update student details
❌ Delete students (with related logs handled)
🖱️ Clickable table for easy update/delete (no manual ID input)
💻 User-friendly Swing-based interface


🛠️ Technologies Used
Java (Swing) – UI Development
JDBC – Database Connectivity
MySQL – Backend Database
Eclipse IDE – Development Environment


▶️ How to Run
Clone the repository
Import project into Eclipse
Add MySQL Connector/J (JAR) to Build Path
Update database credentials in DBConnection.java
Run MainApp.java

🎯 Usage
Launch the application
Use menu buttons to perform operations
Click on a row in "View Students" to update or delete directly
